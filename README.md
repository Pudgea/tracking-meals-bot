# Diet Bot

Telegram-бот для группового дневника питания. Анализирует фото и описания блюд через Gemini, считает КБЖУ и клетчатку, сохраняет в PostgreSQL.

## Команды

### Питание
| Команда | Описание |
|---|---|
| `/count` | Ответить на сообщение с едой — анализ КБЖУ с разбивкой по ингредиентам |
| `/summary` | Сводка по калориям и БЖУ за сегодня |
| `/analyze` | Развёрнутый анализ дня: питание, тренировки, баланс БЖУ, рекомендации |
| `/ask <вопрос>` | Вопрос о питании по контексту дня. Пример: `/ask что съесть сейчас?` или `/ask банан + курица — сколько калорий?` |

### Профиль
| Команда | Описание |
|---|---|
| `/profile` | Показать профиль (рост, цель) |
| `/set_profile <рост_см> [цель]` | Задать рост и цель. Пример: `/set_profile 180 Похудеть до 75 кг` |

### Тренировки
| Команда | Описание |
|---|---|
| `/workout` | Выбрать тренировку из сохранённых шаблонов (кнопки) |
| `/workout <описание>` | Записать произвольную тренировку |
| `/save_workout <название> \| <описание>` | Сохранить шаблон тренировки |
| `/my_workouts` | Список сохранённых шаблонов |
| `/del_workout <id>` | Удалить шаблон |

### Вес
| Команда | Описание |
|---|---|
| `/weight <кг>` | Записать вес на сегодня. Пример: `/weight 74.2` |
| `/weight_stats` | История веса за последние 30 дней с трендом |

## Запуск

```bash
cp .env.example .env
# Заполнить BOT_TOKEN, GEMINI_API_KEY, POSTGRES_PASSWORD в .env

docker compose up --build -d
```

## Переменные окружения

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather |
| `GEMINI_API_KEY` | API-ключ Google Gemini |
| `DATABASE_URL` | Строка подключения к PostgreSQL |
| `POSTGRES_PASSWORD` | Пароль для postgres-контейнера |

Получить ключ Gemini: https://aistudio.google.com/app/apikey
